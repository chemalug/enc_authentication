REST Server API - Encodely

Hola, este es el sitio donde las app de Encodely esta sriviendo los API Rest para el uso de su plataforma:

Haz clic en los enlaces para comprobar si el servidor REST funciona.

Operacion Autenticación

Método POST

  1. {{base_url()}}auth/login - Ingreso - recive los parametros: email y password
  2. {{base_url()}}auth/registrar - Registro - recive los parametros: nombres, email y password

Método GET

  1. {{base_url()}}auth/logout - Cierre de sesión - sin parámetros de envio

Operacion Aerolineas

Método POST

  1. {{base_url()}}aerolinea/insert - Insertar registro - recive los parametros: nombre_aerolinea y ciudad_aerolinea
  2. {{base_url()}}aerolinea/update - Actualizar registro - recive los parametros: id, nombre_aerolinea y ciudad_aerolinea

Método GET

  1. {{base_url()}}aerolinea/show_by/{$id} - Mostrar registro por id - recive el parámetro: id (Primary Key)
  2. {{base_url()}}aerolinea/show_all - Mostrar todos los registros - sin parámetros de envio

Operacion Hoteles

Método POST

  1. {{base_url()}}hotel/insert - Insertar registro - recive los parametros: definidos en la tabla - id
  2. {{base_url()}}hotel/update - Actualizar registro - recive los parametros: definidos en la tabla + id
  3. {{base_url()}}hotel/show_many_by - Mostrar los registros segun por el campo de busqueda - recive los parametros: key =>(campo a buscar) y value=>(condicion del campo)

Método GET

  1. {{base_url()}}hotel/show_by/{$id} - Mostrar registro por id - recive el parámetro: id (Primary Key)
  2. {{base_url()}}hotel/show_all - Mostrar todos los registros - sin parámetros de envio
